
# Controlled Vocabularies  
List of controlled vocabularies used in the FReSH model.    

## Available Information  
For each vocabulary, the following information is provided:  
  - A list of the metadata for which they are used,  
  - A comprehensive list of terms and their definitions in French and English,  
  - Alignment of terms with their equivalents in external vocabularies, where possible,  
  - Access to the RDF/XML file of each vocabulary.
